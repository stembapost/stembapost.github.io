StembaPost
Website Application

StembaPost adalah aplikasi lokal berbasis web yang dibangun dengan interface Framework PHP CodeIgniter, Bootstrap 3, jQuery, HTML 5, dan CSS-Saas. Untuk backend nya digunakan DBMS MySQL (PhpMyAdmin), dan Restful API sebagai connector dengan API WhatsApp.

  • PHP CodeIgniter Boostrap Node.js

Aim and Purpose

Aplikasi ini dikembangkan dengan tujuan sebagai produk karya Tugas Akhir dari dua siswi kelas XIII TKJ 1 yang diharapkan mampu memudahkan sekolah untuk memberikan fitur broadcasting via WhatsApp untuk mengirimkan informasi yang terkini secara cepat dan efisien.

StembaPostwebsite application

Our Hopes

Kami harap, fitur-fitur yang tersedia di Aplikasi STEMBAPOST dapat bermanfaat dan menjadi solusi yang efektif dari permasalahan yang hadir di sekolah, khususnya di SMK Negeri 7 Semarang. Selain itu, kami juga berharap adanya Generasi Penerus yang dapat mengembangkan Aplikasi STEMBAPOST menjadi lebih hebat dan canggih mengikuti perkembangan zaman millenial ini.
